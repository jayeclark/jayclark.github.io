Fun With Arrays

Practice sorting, filtering, mapping, and reducing arrays.

Additional methods or operators that are useful to solve each challenge are identified next to it! Almost all of these can also be solved with basic for loops.

Basic

  • UEFA EURO 2016 Accessing array values, making strings, basic if/then logic... this is a great one to start with!

Array.push()

  • Push an Object into an Array. Ignore the word 'hash' (that's only relevant to people solving it using a different language).

Array.sort()

  • Find the Difference in Age between Oldest and Youngest Family Members.
  • Sort and Star. The Array.join() method is helpful to solve this (and helpful in general!).

Array.filter()

  • You Only Need One. Also use Array.length for this one.
  • Filter Out The Geese. You'll also need to learn what Array.includes() does, but it's simple (and very useful!)
  • Find Numbers Which are Divisible by Given Number. The % operator is useful here, or you could use Math.floor().
  • Counting Sheep. Also use Array.length for this one.
  • Count of Positives / Sum of Negatives The Array.reduce() method is helpful to solve this.

Array.reduce()

  • Check the Exam. You will either need to use Array.map(), OR use TWO arguments in Array.reduce(), the first being the reducing function and the second being the initialValue (used if you are reducing an array where the starting element's value isn't a number.)
  • Count of Positives / Sum of Negatives The Array.filter() method is useful on this one.

Array.map()

  • Lost without a Map.
  • To Square(root) or not to Square(root) You'll also need to learn what the % operator does, as well as the ** operator or Math.sqrt(). This one looks like it may have been written by a non-native English speaker, so let me know if you need help with what it's asking you to do!
  • Take an Arrow to the Knee, Functionally You'll also need to learn what String.fromCharCode() does in order to solve this.
