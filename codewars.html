Fun With Arrays

Practice sorting, filtering, mapping, and reducing arrays.

Additional methods or operators that can be useful to complete each exercise are identified next to it. Almost all of these can also be solved with basic for loops. Try practicing them that way first. And then once you have solved it, you can explore the other method listed!

Array.push()

  • Push an Object into an Array. Ignore the word 'hash' (that's only relevant to people solving it using a different language).

Array.sort()

  • Find the Difference in Age between Oldest and Youngest Family Members.
  • Sort and Star. The Array.join() method is helpful to solve this (and helpful in general!).

Array.filter()

  • You Only Need One. Also use Array.length for this one.
  • Filter Out The Geese. If you learn what Array.includes() does, it will be very useful!
  • Find Numbers Which are Divisible by Given Number. The % operator is useful here, or you could use Math.floor().
  • Counting Sheep. Also use Array.length for this one.
  • Count of Positives / Sum of Negatives The Array.reduce() method is helpful to solve this.

Array.reduce()

  • Reduce but Grow.
  • Check the Exam. You can do this one with just a plain for loop, but if you want to do it in fewer lines of code you could use Array.map(), OR pass TWO arguments to Array.reduce(), the first being the reducing function and the second being the initialValue (used if you are reducing an array where the starting element's value isn't a number.)
  • Count of Positives / Sum of Negatives The Array.filter() method is useful on this one.

Array.map()

  • Lost without a Map.
  • To Square(root) or not to Square(root) You'll also need to learn what the % operator does, as well as the ** operator or Math.sqrt(). The grammar and word choice in this one is a little confusing, so let me know if you need help with what it's asking you to do!
  • Take an Arrow to the Knee, Functionally You'll also need to learn what String.fromCharCode() does in order to solve this.
